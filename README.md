
Professional Make-up Artist Portfolio & Booking System
--------------------------------------------------------

Questa è una web application Full Stack progettata per una Make-up Artist. Permette di mostrare il proprio portfolio di lavori, gestire categorie e ricevere richieste di prenotazione via email. Il sistema include un'area riservata (Admin) per la gestione dei contenuti e un'area utente per la gestione del profilo.

🚀 Tecnologie Utilizzate

Frontend
    - Frontend
    - React.js (Vite)
    - Tailwind CSS 
    - React Router Dom per la navigazione
    
Backend
    - Java 17+ con Spring Boot 3
    - Spring Security & JWT (JSON Web Token) per l'autenticazione sicura
    - Spring Data JPA per la gestione del database
    - MySQL come database relazionale
    - Jakarta Mail per l'invio automatico delle notifiche di prenotazione

  🛠️ Funzionalità Principali
  
Area Pubblica
    - Hero Section: Intro con video background.
    - Portfolio Dinamico: Visualizzazione dei lavori filtrabili per categoria (es. Sposa, Moda, Eventi).
    - Servizi: Listino prezzi e dettagli dei trattamenti offerti.

Area Utente (Guest)
    - Autenticazione: Login e Registrazione sicura.
    - Prenotazioni: Possibilità di richiedere un appuntamento selezionando data e servizio.
    - Gestione Profilo: Aggiornamento dei propri dati (email, username, password).

Area Amministratore (ADMIN)
    - Dashboard Portfolio: Upload di nuovi lavori (immagini) con titolo e descrizione.
    - Gestione Categorie: Creazione e modifica delle categorie dei lavori.
    - CRUD Completo: Modifica ed eliminazione dei lavori esistenti direttamente dal frontend.
